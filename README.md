## Overview

This repository documents two generations of multi-agent systems:

### Generation 1: Multi-Agent Workflow
Specialized agents (Ubuntu, Lisa, Nyx, Kael) collaborating via session handoff and bridge communication.

**Key Features:**
- Session handoff database
- Bridge file communication
- Discord/Telegram/CLI integration
- Platform-specific specialization

### Generation 2: Autonomous Federation v2.0
A unified, self-improving 24-hour autonomous system using 2026 AI patterns.

**Key Features:**
- Semantic routing with VCVs
- Metacognitive self-improvement
- Human-like System 1/2 thinking
- MemoryOS hierarchy
- 24 checkpoints completed
